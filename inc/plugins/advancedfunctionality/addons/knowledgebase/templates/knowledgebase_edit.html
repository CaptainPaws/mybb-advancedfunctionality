Шаблоны

Заполнит примеры текста, meta JSON и создаст типовые блоки.

Дополнительные блоки (Blocks)

Блоки — это дополнительные секции внутри записи. Они помогают структурировать контент: «Бонусы», «Лор», «Способности», «Ограничения», «Примеры», «Синергии».

Блоки можно сортировать, включать/выключать и добавлять сколько нужно.

block_key: bonuses
title_ru/title_en: Бонусы / Bonuses
content_en: +2 to...
data_json:
{
  "grants": ["darkvision"],
  "modifiers": [
    { "stat": "dex", "value": 2 }
  ]
}

Связи

Связи — это связи между сущностями KB. Например: раса → доступные классы; класс → стартовые скиллы; скилл → улучшения; предмет → требования.

rel_type — это название типа связи (например «available_classes», «requires», «upgrades_to»). to_type/to_key — ссылка на другую запись.

rel_type: available_classes
to_type: class
to_key: soldier
meta_json:
{
  "note": "Common for humans",
  "min_level": 1
}
