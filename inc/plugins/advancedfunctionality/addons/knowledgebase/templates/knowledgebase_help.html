Категории и записи

Category/Type — это вкладка или тип сущности (race, class, skill). Она задаёт визуальный стиль (фон, иконку, описание) и группирует записи.

Entry — это конкретная запись внутри категории (например, race → human).

Как выбирать key

Ключ — это уникальный идентификатор внутри категории. Не повторяйте тип в ключе: если тип уже race, то ключ должен быть human, а не race_human.

Блоки (Blocks)

Блоки — это дополнительные секции записи. Они помогают структурировать контент: «Бонусы», «Лор», «Способности», «Ограничения».

Примеры блоков:

  • race → blocks: bonuses, lore, traits
  • class → blocks: role, abilities, progression
  • skill → blocks: rules, effects, examples

Связи (Relations)

Связи описывают отношения между записями.

  • race → grants → skill:darkvision
  • class → requires → skill:athletics

Traits JSON (af_kb.traits.v1)

Traits описывают постоянные свойства сущности и используются внешними модулями.

{
  "schema": "af_kb.traits.v1",
  "traits": [
    {
      "key": "humanoid",
      "title_ru": "Гуманоид",
      "title_en": "Humanoid",
      "desc_ru": "Обычная биология, двуногая морфология.",
      "desc_en": "Standard biology, bipedal morphology.",
      "tags": ["type"],
      "meta": {}
    }
  ]
}

Правила: key — только [a-z0-9_], хотя бы один из title_ru/title_en обязателен.

Grants JSON (af_kb.grants.v1)

Grants описывают выдачу бонусов/эффектов для калькуляторов и карточек персонажей.

{
  "schema": "af_kb.grants.v1",
  "grants": [
    {"op":"stat_add","stat":"str","value":2,"note_ru":"Сила +2","note_en":"Strength +2"},
    {"op":"hp_add","value":4,"note_ru":"Базовые ОЗ +4","note_en":"Base HP +4"},
    {"op":"tag_add","tag":"darkvision","value":true,"note_ru":"Тёмное зрение","note_en":"Darkvision"}
  ]
}

Поддерживаемые op: stat_add, hp_add, skill_add, resist_add, tag_add, grant_kb.

Meta JSON

Meta JSON хранит технические данные (grants, modifiers, tags, ui). Примеры:

{
  "ui": {
    "icon_url": "https://site/icons/races/human.png",
    "background_url": "https://site/bg/races.jpg"
  },
  "grants": ["darkvision"],
  "modifiers": [
    {"stat":"dex","value":2}
  ],
  "tags": ["civilized","starter"]
}
{
  "ui": {
    "icon_url": "https://site/icons/skills/stealth.png"
  },
  "type": "active",
  "cost": 2,
  "cooldown": 1,
  "effects": [
    {"kind":"buff","stat":"dex","value":1,"duration_turns":3}
  ]
}

Пример для relation meta_json:

{
  "reason": "racial_trait",
  "note": "Granted at character creation"
}
